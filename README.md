# Simple Interest Calculator in C

## 📌 Description
This project calculates simple interest using the formula:
SI = (P × R × T) / 100

## 🛠️ Technologies Used
- C Programming Language

## ▶️ How to Run
1. Compile:
   gcc main.c -o si
2. Run:
   ./si

## 📊 Example
Input:
Principal = 1000
Rate = 5
Time = 2

Output:
Simple Interest = 100.00
